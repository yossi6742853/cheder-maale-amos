צעד 3: עבודה

המסך הראשי מציג כפתורים גדולים לכל הפעולות. לחיצה על כפתור = פתיחת המודול.

👨‍🎓 ניהול תלמידים

➕ הוספת תלמיד חדש (אחד-אחד)

  1. לחצו על הכפתור “תלמידים” בדף הבית.
  2. למעלה מימין יש כפתור “➕ תלמיד חדש” - לחצו עליו.
  3. ייפתח טופס. מלאו:
    • שם פרטי (חובה)
    • שם משפחה (חובה)
    • גיל
    • כיתה - בחרו מהרשימה (א / ב / ג / ד / ה / ו / ז / ח)
    • שם אם + טלפון אם
    • שם אב + טלפון אב
    • כתובת
    • 🔴 אלרגיה / רגישות - אם יש משהו, חובה לכתוב כאן! (לדוגמה: בוטנים, אבק, ביצים)
    • הערות
  4. לחצו “שמור”.

✅ הסטטיסטיקה בדף הבית תתעדכן מיד.

📥 הוספת המוני תלמידים יחד

אם יש לכם רשימה של 20-30-100 תלמידים בקובץ Excel או ב-Google Sheets - אפשר להעלות הכל בלחיצה אחת.

דרך 1: קובץ Excel / CSV

  1. לחצו “תלמידים” → “📥 ייבוא CSV”.
  2. ייפתח חלון עם שתי לשוניות. השאירו את הלשונית הראשונה “📁 קובץ”.
  3. לחצו “בחירת קובץ” ובחרו את ה-.xlsx או .csv שלכם.
  4. תוך שנייה תופיע תצוגה מקדימה - תראו 5 שורות ראשונות וכמה תלמידים זוהו.
  5. בדקו שזה נראה תקין.
  6. למטה יש כפתור כחול גדול: “ייבא X תלמידים” - לחצו.
  7. תוך כמה שניות יסתיים. תקבלו הודעה “✅ נוספו X תלמידים”.

דרך 2: הדבקה ישירות מ-Google Sheets

זו הדרך הכי קלה אם הנתונים כבר נמצאים ב-Sheet.

  1. פתחו את ה-Sheet עם רשימת התלמידים (בדפדפן או בטלפון).
  2. סמנו את כל השורות שאתם רוצים (כולל שורת הכותרות).
  3. לחצו Ctrl+C (Mac: Cmd+C).
  4. חזרו לאתר → “תלמידים” → “📥 ייבוא CSV”.
  5. בחרו את הלשונית השנייה: “📋 הדבקה מ-Google Sheets”.
  6. לחצו בתיבת הטקסט הגדולה והקליקו Ctrl+V. הטקסט יודבק שם.
  7. לחצו על הכפתור “⬇️ זיהוי שורות מהטקסט”.
  8. תופיע תצוגה מקדימה כמו בדרך הראשונה.
  9. לחצו “ייבא X תלמידים”.

💡 חשוב: השורה הראשונה חייבת להיות כותרות עמודות. מילים כמו “שם פרטי”, “שם משפחה”, “גיל” וכו’.

📝 עמודות מומלצות: שם פרטי, שם משפחה, גיל, מחזור, תאריך לידה, שם אם, טלפון אם, שם אב, טלפון אב, כתובת, אלרגיה, הערות

✏️ עריכת תלמיד / כרטיס תלמיד

  • לחצו על שם התלמיד ברשימה → ייפתח כרטיס מלא עם כל הפרטים.
  • בכרטיס תראו: פרטים, אירועי התנהגות אחרונים, מעקב רפואי, שיחות, ועוד.
  • כפתור “📝 עריכה” למעלה - לפתיחת הטופס לעריכה.
  • כפתור WhatsApp ירוק ליד מספר הטלפון - שליחת הודעה ישירה לאמא או אבא.

📋 מעקב התנהגות

➕ אירוע התנהגות חדש

  1. “מעקב התנהגות” → “➕ אירוע חדש”.
  2. בחרו תלמיד מהרשימה.
  3. בחרו קטגוריה:
    • התנהגות (בכיתה)
    • דיבור עם הורים
    • אסיפת הורים
    • מוגנות
    • חינוך אישי
    • חברה
    • לימודים
    • בריאות
  4. בחרו חומרה: 🟢 נמוכה / 🟡 בינונית / 🔴 גבוהה.
  5. כתבו תיאור מפורט.
  6. לחצו “שמור”.

📊 הסטטיסטיקה תתעדכן אוטומטית והאירוע יופיע ב”פעילות אחרונה” בדף הבית.

🏥 רפואי - אלרגיות, רגישויות, כדורים

זה המודול שמנהל את כל המידע הרפואי על התלמידים - לא רק כדורים.

➕ רישום רפואי חדש

  1. “רפואי” → “➕ רישום חדש”.
  2. בחרו תלמיד.
  3. בחרו סוג:
    • 💊 תרופה (ריטלין / קונסרטה / וכו’)
    • ⚠️ אלרגיה - חמורה! חובה לרשום מה (בוטנים / גלוטן / אבק)
    • 🌿 רגישות - קלה יותר
    • 📋 מעקב רפואי כללי
    • 📌 אחר
  4. פירוט: מה התרופה? מה החומר האלרגן? לדוגמה: “בוטנים” / “ריטלין 10 מ”ג”
  5. מצב כיום: לדוגמה: “1 כדור בוקר ו-1 כדור בצהריים”. או: “רגישות חמורה - לא לתת בוטנים בכלל!”
  6. שיחת הורים: מתי דיברתם לאחרונה, מה סוכם.
  7. הערות.
  8. “שמור”.

🚨 בכרטיס התלמיד יוצג בולט באדום אם יש אלרגיה. אל תתעלמו מזה.

✅ נוכחות

  1. “נוכחות” → בחרו כיתה ותאריך.
  2. לכל תלמיד סמנו: 🟢 נוכח / 🔴 נעדר / 🟡 חולה.
  3. שומרים.

💡 ניתן לראות סטטיסטיקת נוכחות חודשית של כל תלמיד בכרטיס שלו.

💬 שיחות עם תלמידים

  1. “שיחות” → “➕ שיחה חדשה”.
  2. בחרו תלמיד.
  3. נושא השיחה.
  4. רב מטפל.
  5. סיכום מפורט.
  6. “שמור”.

⚠️ המערכת תתריע על תלמידים שלא דיברו איתם הרבה זמן.

👨‍👩‍👦 אסיפות הורים

תיעוד פגישות עם הורים.

  1. “אסיפות” → “➕ אסיפה חדשה”.
  2. תלמיד, תאריך, נושא, משתתפים.
  3. סיכום הפגישה.
  4. הערות + תאריך לפגישה הבאה.

🔔 המערכת מזכירה אסיפות שצריך לקיים בעוד יום-יומיים.

📝 מבחנים

  1. “מבחנים” → “➕ מבחן חדש”.
  2. בחרו פרשה / נושא.
  3. מזינים ציון לכל תלמיד.
  4. שומרים.

📈 בכרטיס תלמיד תראו גרף ציונים לאורך זמן.

📄 דוחות PDF

  1. “דוחות” → בחרו סוג:
    • דוח תלמיד יחיד - כל המידע על תלמיד מסוים.
    • דוח כיתה שלמה - סיכום פעילות הכיתה.
    • דוח בית ספר - סטטיסטיקה כללית.
  2. בחרו תקופה (חודש / רבעון / שנה).
  3. לחצו “📥 הורד PDF”.

💼 הדוח מקצועי, מוכן להדפסה או שליחה במייל.

⚙️ הגדרות (אדמין בלבד)

ניהול משתמשים

  • “הגדרות” → “משתמשים”.
  • ➕ משתמש חדש: שם משתמש, סיסמה ראשונית, תפקיד (מנהל / רב / מורה), הרשאות.
  • 🔑 איפוס סיסמה: אם משתמש שכח - אדמין יכול ליצור לו סיסמה חדשה.
  • 🚫 הסרת משתמש - מי שכבר לא עובד.

ניהול קטגוריות התנהגות

  • מוסיפים / משנים שמות קטגוריות.

ניהול כיתות

  • מוסיפים / משנים שמות כיתות.

💡 טיפים שווה לדעת

קיצורי דרך

  • / - חיפוש מהיר בכל המערכת (תלמיד / אירוע).
  • Esc - חזרה לדף הבית.
  • Ctrl+F5 - רענון מלא. אם משהו לא נראה מעודכן - תמיד תנסו את זה קודם.

מצב לילה (כהה)

  • בפינה ימנית למעלה יש 🌙 - לחיצה תהפוך את האתר לכהה. נוח לעיניים.

גיבוי

  • הכל נשמר אוטומטית בשיטס. גם אם המחשב יורד.
  • בכל זמן אפשר לפתוח את ה-Sheet ישירות מ-Google Drive ולראות / לעדכן.

מובייל

  • האתר עובד מצוין בטלפון ובטאבלט.
  • ניתן להוסיף אירועים מהר מהשטח.

❓ אם משהו לא עובד

  1. רעננו את הדף - Ctrl+F5 (Windows) או Cmd+Shift+R (Mac).
  2. נקו cache - לפעמים הדפדפן מציג גרסה ישנה.
  3. חיבור אינטרנט - האתר זקוק לחיבור פעיל.
  4. שלחו לי מייל - אם לא נפתר, כתבו לי בדיוק:
    • באיזה עמוד הייתם
    • על מה לחצתם
    • מה ראיתם (אפשר צילום מסך)
